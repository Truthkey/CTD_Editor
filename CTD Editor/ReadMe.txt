========================================================================
    APLICACIÓN: Información general del proyecto CTD Editor
========================================================================

AppWizard ha creado esta aplicación CTD Editor.  

Este archivo incluye un resumen acerca del contenido de los archivos que
constituyen su aplicación CTD Editor.

CTD Editor.vcproj
    Éste es el archivo de proyecto principal para los proyectos de VC++
    generados mediante un Asistente para aplicaciones. 
    Contiene información acerca de la versión de Visual C++ con la que
    se generó el archivo, así como información acerca de las plataformas,
    configuraciones y características del proyecto seleccionadas en el
    Asistente para aplicaciones.

CTD Editor.cpp
    Ésta es la aplicación principal del archivo de código fuente.
    Contiene el código para mostrar el formulario.

Form1.h
    Contiene la implementación de su clase de formulario y función 
    InitializeComponent().

AssemblyInfo.cpp
    Contiene atributos personalizados para modificar metadatos de 
    ensamblado.

/////////////////////////////////////////////////////////////////////////////
Otros archivos estándar:

StdAfx.h, StdAfx.cpp
    Estos archivos se utilizan para generar un archivo de encabezado 
    precompilado (PCH) denominado CTD Editor.pch y un archivo de 
    tipos precompilado denominado StdAfx.obj.
